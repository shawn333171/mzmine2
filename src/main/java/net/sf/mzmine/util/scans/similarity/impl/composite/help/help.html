Composite weighted cosine spectral similarity (identity)

Description

Is used to determine the similarity between two spectra (usually library and query spectra). Especially useful for very reproducible generation of spectra (GC-EI-MS). Takes into account the relative intensities of neighbouring signals in the two spectra.

Method parameters

Weights
Weights are applied to m/z and intensity: m/za*Intensityb
Minimum cos similarity
Minimum cosine similarty for a match
